第四章、首次登入与在线求助

2022-03-14 鸟哥的Linux私房菜-读书笔记
  • 首次登入系统
    • 首次登录CentOS 7.x系统注意事项

      • 若安装时没有设置切换输入法的快捷键，那么[设置–>键盘]可设置

      • 调整背光时间，那么[设置–>电源]可设置

      • 设置工作区数量及CTRL+ALT+backspace重启动x窗口，那么[应用程序–>工具–>优化工具]

      • x window与文本模式切换：

        1.LANG=en_US.UTF8,LANG只与输出信息有关
        2.export LC_ALL=en_US.UTF8，如需要更改其他不同的信息，需同步更新LC_ALL

      • 基础指令

        1. TAB：
        • 接在一串指令的第一个字后面：命令补全
        • 接在一串指令的第二个字后面：文件补齐
        • 安装bash-completion软件，在某些指令后面使用tab键可以补齐选项或参数
        1. CTRL+c：终端目前程序
        2. CTRL+d：约等于exit，例如直接离开文字接口
  • Linux系统的man和info
    • 帮助与关机
      • 查询你曾经用过的指令所具备的选项和参数：–help
      • 查询命令的用法与相关说明：man

      • 鸟哥的查询步骤
      1. options，-f，查询与某指令相关的说明文件[只会找指令或文件的完整名称]，eg：man -f man，相当于apropos [指令或数据]
      2. options，-k，不同于-f，可找关键词，相当于whatis [指令或数据]
      3. man的搜寻顺序：记录在/etc/man_db.conf这个配置文件中，先搜寻到的说明文件会被显示出来
      4. 注：_**要想使用whatis和apropos，需要有建立whatis数据库，使用mandb，旧版Linux使用makewhatis

      1.who：目前谁在线
      2.netstat -a：网络的联机状态
      3.ps -aux：后台执行的程序
      *将数据同步写入硬盘：sync，[因为你重复使用的数据被放进内存里了呀]

    • 关机指令：shutdown eg：systemctl reboot
  • 重点回顾
    • 重启x的组合按键：ALT+CTRL+backspace，记得要开启才行
    • man page说明后面的数字中，1表示一般账号，8表示系统管理员常用命令，5表示系统配置文件

第二章、主机规划与磁盘分区

2022-03-14 鸟哥的Linux私房菜-读书笔记
  • #磁盘分区

    • ##磁盘连接的方式与装置文件名的关系

      • 正常的实体机器大概使用的都是 /dev/sd[a-] 的磁盘文件名，至于虚拟机环境底下，为了加速，可能就会使
        用 /dev/vd[a-p] 这种装置文件名

      • SATA/USB 接口的磁盘根本就没有一定的顺序，那如何决定他的装置文件名呢？
        根据 Linux 核心侦测到磁盘的顺序来决定文件名,这里鸟哥举了一个例题，“两个SATA硬盘和一个U盘插在主板上，SATA硬盘分别插在SATA1、SATA5上，问三个磁盘在linux中的文件名”，答“和插槽号没有关系,根据 Linux 核心侦测到磁盘的顺序来决定文件名,所以分别是/dev/sda、/dev/sdb、/dev/sdc(开机完成后才被捕捉到)”

      • 磁盘盘上面又可细分出扇区(Sector)与磁道(Track)两种单位， 其中扇区的物理量设计有两种大小，分别是 512bytes 与 4Kbytes。
